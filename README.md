## Moonlet Validators

Setup mainnet and testnet blockchain nodes using Docker.

### Commands

1. Clone repo

```
https://github.com/Moonlet/validators.git
cd validators
chmod +x build.sh run.sh
```

2. Update the `env` file

```
vi env
```

3. Build the Docker image

```
./build.sh
```

4. Run the Docker image

```
./run.sh
```
